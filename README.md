# Klivio Project

*This project is part of my learning journey as a web developer.*
---

## 📌 About the project (english version🇬🇧)
This project was developed as part of my training at Web@cademie by EPITECH.

The objective was to reproduce a web interface from a given design (mockup) and practice web integration skills.

## 🎨 Project context
This project focuses on **front-end integration** based on a provided mockup.

- The reference design and all images are available in the `images` folder.
- Two versions of the project were developed:
  - A version using **HTML/CSS**
  - A version using **Tailwind CSS**

## 🔗 Live demo
- HTML/CSS version:  
  https://dalobaminthe.github.io/Projet-Klivio/html-css/

- Tailwind version:  
  https://dalobaminthe.github.io/Projet-Klivio/tailwind/

## 🗂️ Project structure
The repository contains:
- `html-css` folder → classic integration using HTML & CSS  
- `tailwind` folder → integration using Tailwind CSS  
- `images` folder → assets used for the project  

## 🛠️ Technologies used
- HTML  
- CSS  
- Tailwind CSS  

## 🎯 What I learned
- Reproducing a design from a mockup  
- Structuring a front-end project  
- Understanding layout and responsiveness  
- Improving code organization and readability  

## 🚀 Project status
This is a training project created during my learning journey.  
I am continuously improving my skills through new projects.

## 📎 Notes
This project was completed as part of a school assignment.
